# Data Files (Not Included in Git)

The following data files are excluded from Git due to size constraints:

## Excel Files
- `NLL_Analytics_FINAL.xlsx` (17 MB) - Primary NLL dataset
- `NLL_Data_2020_2024_V2.xlsx` (5.7 MB) - Alternative dataset version

## Generated Data (nll_betting_analysis/data/)
These files are generated by running the analysis scripts:

### To regenerate:
```bash
# 1. Load data from Excel
python nll_betting_analysis/scripts/data_loader.py

# 2. Generate features
python nll_betting_analysis/scripts/feature_engineering.py

# 3. Train models
python nll_betting_analysis/scripts/models.py
```

### Files generated:
- `raw_data.json` (7.2 MB) - Raw data from Excel
- `processed_matches.json` (548 KB) - Processed match records
- `features.json` (2 MB) - Engineered features (62 features)
- `team_stats_by_match.json` (452 KB) - Team statistics
- `standings_lookup.json` (36 KB) - Weekly standings
- `analysis_summary.json` (8 KB) - Analysis results
- `model_results.json` (4 KB) - Model performance metrics

## Setup Instructions

1. Place your Excel files in the root directory
2. Install dependencies: `pip install -r requirements.txt`
3. Run the scripts above to generate the data files
4. Models will be trained and ready for predictions

## Note
The Excel files contain NLL match data from 2021-2025 seasons (567 matches total).
